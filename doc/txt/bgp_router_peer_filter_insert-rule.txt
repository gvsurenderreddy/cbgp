[1mNAME[0m
     [1mbgp router peer filter insert-rule [22m-- insert a rule in a filter

[1mSYNOPSIS[0m
     [1minsert-rule [22m<[4mindex[24m>

[1mARGUMENTS[0m
     <[4mindex[24m> index where rule should be inserted

[1mDESCRIPTION[0m
     This command will insert a new rule at the [4mindex[24m specified as argument.
     The predicate and actions of the new rule can then be defined using com-
     mands [1mmatch [22mand [1maction [22mrespectively.

[1mSEE ALSO[0m
     The command [1madd-rule [22mcan be used to append a rule.
