[1mNAME[0m
     [1mnet node [22m-- enter one node's context

[1mSYNOPSIS[0m
     [1mnode [22m<[4maddr[24m>

[1mARGUMENTS[0m
     <[4maddr[24m> node's address (identifier)

[1mDESCRIPTION[0m
     Strictly speaking, this is not really a command. It rather allows you to
     enter one node's context and enter commands that are specific to this
     context.
