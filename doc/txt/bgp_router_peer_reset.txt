[1mNAME[0m
     [1mbgp router peer reset [22m-- reset a BGP session

[1mSYNOPSIS[0m
     [1mreset[0m

[1mDESCRIPTION[0m
     This command resets a BGP session. This is equivalent to stopping the BGP
     session with command [1mdown [22mand immediately re-starting it using command [1mup[0m
     .
