[1mNAME[0m
     [1mbgp show route-maps [22m-- list all availabel route-maps

[1mSYNOPSIS[0m
     [1mroute-maps[0m

[1mDESCRIPTION[0m
     This command lists all defined route-maps.

[1mSEE ALSO[0m
     To define a route-map, use command [1mbgp route-map [22m.
