[1mNAME[0m
     [1msim step [22m-- process a single simulation event

[1mSYNOPSIS[0m
     [1mstep [22m<[4mnum-steps[24m>

[1mARGUMENTS[0m
     <[4mnum-steps[24m>

[1mDESCRIPTION[0m
     This command processes a limited number of events from the pending event
     set.

[1mSEE ALSO[0m
     To process all events until there is no more pending event, you can use
     command [1msim run [22m.
